# petstagram

**Petstagram is a Social media app for pets. Where users can browse to see the most beautiuful pets or if they register they can upload their own pets or like and comment other pets or edit their pets.**

## Technologies used:
1. Python,
2.  Javascript
3.  Django
4.   HTML
5.    CSS
6.    Bootstrap
7.    Django-tests
